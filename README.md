## Description

{Ronin::SQL} is a Ruby DSL for crafting SQL Injections.

### Features

* Provides an Domain Specific Language (DSL) for crafting normal SQL and
  SQL injections.

## Requirements

* [ronin-support] ~> 0.2

## Install

    $ gem install ronin-sql
